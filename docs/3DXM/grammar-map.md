|---|---|---|---|---|

| **Surfaces of Revolution** | Revolution surfaces | S(r,z) = 0, generated by profile curve | `surface.revolution` | Sphere, torus, cylinder, cone |

| **Ruled Surfaces** | Ruled/generator | Linear combination of two curves | `surface.ruled`, `surface.generator` | Pseudosphere, conoids |

## Grammar Extension Recommendations

### Add to OctaneX Scene Construction Pipeline

1. **Minimal Surface Grammar**: Add `surface.implicit` type with Z-f(X,Y) parameterization support
   - Use: `surface.geometry_type = 'implicit'` 
   - Add: `surface.curvature = 0` for minimal surfaces (H=0)
   - Examples: Catenoid (H=0), helicoid (H=0), Scherk surface (H=0)

2. **Torus Knot Extension**: Add `curve.torus_knot` with (p,q) parameter support
   - Use: `curve.torus_knot = True`
   - Add: `curve.winding_numbers = [p, q]` for torus knot specification
   - Examples: (2,3)-torus knot, (3,4)-torus knot, p/q rational knots

3. **Fractal Basin Mapping**: Add `fractal.basin` with iteration basin depth
   - Use: `fractal.basin_depth = n` for M-basins
   - Add: `fractal.convergence_threshold = ε` for basin boundary precision
   - Map: Mandelbrot set to `fractal.basin_depth = 100`, Julia sets by orbit period

4. **Conformal Grid Support**: Add `conformal.grid_map` for complex plane transformations
   - Use: `grid_map.transformation = 'mobius' | 'elliptic' | 'log' | 'exp'`
   - Add: `grid_map.parameter_k` for elliptic modulus (Jacobi functions)
   - Map: Complex grid → surface texture/point cloud

5. **Spherical Surface Curvature**: Add `surface.curvature = K` for constant Gaussian curvature
   - Use: `surface.curvature = 1.0` for spherical (K=+1)
   - Use: `surface.curvature = -1.0` for pseudospherical (K=-1)
   - Example: Stereographic projection surfaces, constant H (mean curvature)

6. **Polyhedral duality**: Add `solid.dual` type for Platonic pairs
   - Use: `solid.dual_pair = 'cube_octahedron' | 'dodecahedron_icosahedron'`
   - Add: `solid.truncation_level` for Archimedean series
   - Map: Vertex duality to node_graph node_type attribute

7. **Ruled Surface family**: Add `surface.ruled` with generator curve support
   - Use: `surface.ruled = True`
   - Add: `surface.generator_curve = 'cubic' | 'quadratic' | 'transcendental'`
   - Examples: Pseudosphere, right conoid, hyperbolic paraboloid
