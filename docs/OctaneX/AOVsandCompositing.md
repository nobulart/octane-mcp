Octane\'s Arbitrary Output Variables (AOVs) system is extensive and contains two types of AOV systems, the Render AOVs and the Output AOVs. They provide a way to composite the AOVs together directly in OctaneRender®. The AOV systems can be found under the Render AOV and Ouptut AOV categories in the Node Graph Editor window (figure 1). The AOV implementation replaces the Render Pass implementation in older versions of Octane
