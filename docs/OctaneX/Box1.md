### Box Parameters

Width - The width of the box.

Height - The height of the box.

Depth - The depth of the box.

Round Base - Rounds the base where a value of 1 makes the box become a cylinder.

Round - Rounds the edges of the box.

Fill - Can be used to control the edge thickness when two or more Vectron objects are subtracted or intersected with each other.
