### Capsule Parameters

Top Radius - Determines the top radius of the capsule.

Bottom Radius - Determines the bottom radius of the capsule.

Height - Determines the height of the capsule.

Fill - Can be used to control the edge thickness when two or more Vectron objects are subtracted or intersected with each other.
