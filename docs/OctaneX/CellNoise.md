The Cell Noise texture node generates a random checker pattern that has a cellular look when scaled appropriately (figure 1). The parameters consist of UVW Transform and Projection inputs which can be used to edit the pattern\'s position and scale. The RGB Output checkbox toggles between color and greyscale.
