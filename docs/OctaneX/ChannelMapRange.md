The Channel Map Range Output AOV node can be used to remap the input pixel values to specified output pixel value ranges for RGB, Alpha, or RGB+Alpha (figure 1). The Clamp parameter clamps the values to the minimum and maximum range, otherwise, values outside the range will be unmodified.
