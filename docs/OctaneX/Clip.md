### Clip Parameters

Bounds - Determines the area to be clipped, anything outside of the bounds is clipped away.

Input - A Vectron source can be added to this input as the object to be clipped.

Transform - Used to transform the clipping bounds in the scene.

Projection - Used to adjust the projected angle of the clipping bounds.

Inverse - Inverts the clipping bounds.

Radius - Determines how far from the clipping bounds the input object will be clipped.

Show Clipping Plane - Shows or hides the clipping plane.
