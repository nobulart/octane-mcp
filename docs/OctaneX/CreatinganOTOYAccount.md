To create an OTOY® account:

1.  1.  Go to [home.otoy.com](http://home.otoy.com/).
    2.  Click the ![](images/SignInButton.png) Sign-In button at the upper-right corner of the page.
    3.  Click Register A New Account.

+-----------------------------------+------------------------------------------------+
| ![](images/NewItem_506.png)       | New Account                                    |
|                                   |                                                |
|                                   | ![](images/LogInPageWithHighlight_455x501.png) |
+-----------------------------------+------------------------------------------------+

Figure 1: Clicking on Register A New Account

1.  1.  Enter your information in the Sign Up page, then click the Register button.

+-----------------------------------+--------------------------------------+
| ![](images/NewItem_507.png)       | Sign Up Page                         |
|                                   |                                      |
|                                   | ![](images/SignUpScreen_444x524.png) |
+-----------------------------------+--------------------------------------+

Figure 2: Sign Up page

1.  1.  In the Complete Registration form, enter your information in the User Information tab, then click Continue.

+-----------------------------------+-----------------------------------------+
| ![](images/NewItem_508.png)       | Registration Form                       |
|                                   |                                         |
|                                   | ![](images/Customer_Registration_1.png) |
+-----------------------------------+-----------------------------------------+

Figure 3: Complete Registration form

1.  1.  Continue entering your information in the Billing Information tab, then click Continue.

+-----------------------------------+-----------------------------------------+
| ![](images/NewItem_509.png)       | Billing Information                     |
|                                   |                                         |
|                                   | ![](images/Customer_Registration_2.png) |
+-----------------------------------+-----------------------------------------+

Figure 4: Billing Information tab

1.  1.  The following step depends on whether or not you have an Octane forum account.

a. If you do not have a forum account, one is created for you at this time. Enter your forum account password in the Password and Confirm Password fields, then click Continue.

+-----------------------------------+-----------------------------------------+
| ![](images/NewItem_510.png)       | Forum Account                           |
|                                   |                                         |
|                                   | ![](images/Customer_Registration_3.png) |
+-----------------------------------+-----------------------------------------+

Figure 5: Forum Account tab

b. If you do have a forum account, click I Have A Forum Account, enter your forum account password, then click Continue. You can return to the account creation screen by clicking I Don\'t Have A Forum Account.

| ![](images/NewItem_511.png)       | No Account                                |
|                                   |                                           |
|                                   | ![](images/Customer_Registration_3-b.png) |
+-----------------------------------+-------------------------------------------+

Figure 6: I Don\'t Have A Forum Account option

1.  1.  Click I Am Not A Robot in the mCAPTCHA screen.
