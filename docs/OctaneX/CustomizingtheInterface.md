You can rearrange the windows for each pane in the OctaneRender® interface (Graph Editor, Render Viewport, Node Inspector, and Outliner) by clicking and dragging the square at the top-left corner of each pane (Figure 1).

|                                   |                                                          |
|                                   | ![](images/Customizing_The_Interface__Fig01_SE_2026.jpg) |
+-----------------------------------+----------------------------------------------------------+

Figure 1: Pane arrangement icon

Clicking the same square with any mouse button brings up more options for rearranging the workspace. You can undock the Render Viewport window to have a wider view of it on a separate screen (Figure 2).

Figure 2: Undocking the Render Viewport

The toolbars for each pane of the OctaneRender interface are customizable as well. To customize the toolbars, right-click on the square upper-left icon of that pane and select Customize Toolbar (Figure 3).

+-----------------------------------+----------------------------------------------------------+
| ![](images/NewItem_593.png)       | Customize                                                |
|                                   |                                                          |
|                                   | ![](images/Customizing_The_Interface__Fig03_SE_2026.jpg) |
+-----------------------------------+----------------------------------------------------------+

Figure 3: Customize Toolbar option

The dialog lets you place the toolbar to any edge of the respective area, and you can re-arrange, add, or remove buttons.
