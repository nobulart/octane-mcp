### Cylinder Parameters

Radius - Determines the radius of the cylinder.

Height - Determines the height of the cylinder.

Round - Rounds the edges of the cylinder. 

Fill - Can be used to control the edge thickness when two or more Vectron objects are subtracted or intersected with each other.
