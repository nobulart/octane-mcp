You can find the following [Displacement](javascript:void(0);) nodes by right-clicking in the Nodegraph Editor and navigating to the Displacement category:

- Texture Displacement
- Vertex Displacement
- Vertex Displacement Mixer
- Displacement Switch - This node which can be used to connect two or more displacement nodes to the same input pin for switching purposes during the rendering process.
