### Geometric Primitives Parameters

The specific parameters for the geometric primitives will vary depending on which primitive is selected.

#### General Parameters

[Material](javascript:void(0);) - The Material input. See the [Materials](javascript:void(0);) topic in this manual for more details.

Object Layer - The Object Layer input. See the Object Layer Node topic under Rendering for details.

Transform - The Transform input. See the Transforms topic under [Textures](javascript:void(0);) for more details.

Parameters - These will vary depending on the primitive selected.
