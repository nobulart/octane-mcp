### Image File Parameters

Enabled - Determines whether the image file is active or not.

Color Space - Sets the color space for the imported image. 

Premultiplied Alpha - Determines whether the RGB channels have been premultiplied by the alpha channel. 

Blending Settings - Determines the blending mode for the imported image.
