The default interface consists of four main windows (figure 1). The interface is customizable as described in the [Customizing the Interface](CustomizingtheInterface.md) section.

Figure 1: The four primary windows in the Standalone default configuration

1 - The Render Viewport - Updates the render, and you can adjust the materials, lights, and cameras in real-time, along with navigating the scene.

2 - The Node Inspector - From here, you can change the various settings in the scene, lighting, and Material nodes.

3 - The Graph Editor - The scene and materials are represented by nodes, which you can edit from the Graph Editor.

4 - The Scene Outliner - See an overview of all of the scene\'s elements and use the local and online node databases.
