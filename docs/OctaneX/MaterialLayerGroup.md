The [Material](javascript:void(0);) Layer Group node lets you add multiple Material layers to existing Octane materials, without needing a Layered material node. It can connect to the Material layer pin on existing materials.

### Layer Group Parameters

Add Layer - Adds a new Layer input to the end of the node. You can add up to eight Layer inputs to a node.

Layer 1 - 8 - The Material layer input.
