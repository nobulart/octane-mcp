OctaneRender® now supports MaterialX natively (figure 1). MaterialX is a platform-independent  open standard developed by Industial Light & Magic. It is a set of computer graphics standards representing material and look-development content that can be shared across multiple DCC applications and render engines. The MaterialX nodes can be accessed from the Node Graph Editor window (figure 1). For more information refer to the detailed documentation found on [Materialx.org](https://materialx.org/).
