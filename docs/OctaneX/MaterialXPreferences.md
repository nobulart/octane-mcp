OctaneRender® now supports MaterialX natively (figure 1). MaterialX is a platform-independent  open standard developed by Industial Light & Magic. It is a set of computer graphics standards representing material and look-development content that can be shared across multiple DCC applications and render engines. The MaterialX preferences consists of two options: use Octane OSL textures instead of MaterialX native textures and use Octane RGB image nodes instead of MaterialX native images.
