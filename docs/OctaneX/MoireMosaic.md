### Moire Mosaic Parameters

Shape - Determines the type of shape used for the pattern.

Size - Determines the size of the pattern shapes.

Corner Radius - Rounds the corners of the Frame shape.

Blur - Add a blur to the Circle and Ring shapes.

Shift - Applies a horizontal shift to alternating rows when using the rectangle and circle shapes.

Ring Spacing - Specifies the horizontal and vertical spacing when using the Ring and Frame shapes.

Iteration Count - Determines the number of iterations used by the shape generator.

Time - Provides an additional offset value when using the Ring and Frame shapes.

UVW Transform - Positions, scales, and rotates the surface texture.

Projection - Sets how the texture projects onto the surface.
