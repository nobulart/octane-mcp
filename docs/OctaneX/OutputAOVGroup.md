The Output AOV Group node is used to collect multiple Output AOV nodes together and is usually the node plugged into the Output AOV pin on a Render Target node (figure 1).The Output AOV Group node has only two parameters, an Add AOV Output and Remove AOV Output button.

 The Output AOV Group node does not generate a composited result viewable in the Render Viewport. Each set of Output AOV nodes are available for viewing when they are connected together using the Output AOV Group node (figure 2)
