### Prism Parameters

Sides - Determines the number of sides for the prism shape.

Radius - The radius of the prism.

Height - Determines the height of the prism.

Round Base - Rounds the vertical edges without rounding the horizontal edges.

Round - Rounds all edges equally.

Fill - Can be used to control the edge thickness when two or more Vectron objects are subtracted or intersected with each other.
