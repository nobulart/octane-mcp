Projection nodes can add or edit existing UV mapping coordinates on a texture map. These nodes do not alter the actual 3D model\'s UV coordinates - they simply apply a new UV projection to its connected texture in the Nodegraph Editor. This category also contains a Projection Switch node which can be used to connect two or more projection nodes to the same input pin for switching purposes during the rendering process.
