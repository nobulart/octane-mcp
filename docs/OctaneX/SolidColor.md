The Solid Color blend node is used to add a single color to the composite node tree. In figure 1, The Solid Color node is used to add a white background behind the cube and the ground plane which has a Shadow Catcher material applied to it and Alpha Channel activated in the Kernel parameters.

### Solid Color Parameters

Enabled - Determines whether the Solid Color node is active or not.

Color - Specifies the color. 

Alpha - Specifies the level of opacity for the Solid Color.

Blending Settings - Determines the blending mode for the Solid Color.
