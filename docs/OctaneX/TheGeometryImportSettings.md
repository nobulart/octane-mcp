### Geometry Import Settings Parameters

Edit Settings Before Loading New Geometry - When enabled, the Geometry Import Settings dialog appears every time you load a mesh object or an Alembic scene so that you can check these settings and adjust them before loading the scene. You can also make the settings the default and disable this option so that the dialog doesn\'t appear.
