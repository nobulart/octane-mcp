This tab provides access to third-party extensions or small pieces of software called Octane Modules that add more functionality to OctaneRender®. The Octane Modules are created by third-party developers through Octane's Module API. This allows other companies' plugins to work with OctaneRender. Modules are different than plugins, which is already used for integrations of OctaneRender into other 3D modelling applications.
