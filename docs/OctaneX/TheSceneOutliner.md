The Scene Outliner lets you view the current project in an outline format (Figure 1). This provides quick access to any of the scene\'s parameters and nodes. Selecting a parameter in the Outliner brings that node to focus in the Node Inspector. The Scene Outliner also has context menus for copying, pasting, and filling empty Node pins.

The tabs in the Scene Outliner let you view the local and live databases that can house any macro you create.

The Live Database (Live DB tab) allows instantaneous access to any macro that resides on the OctaneRender® community.  The Local Database (Local DB tab) is next to the LiveDB tab. In this tab, you can set the root directory in the application preferences to allow some simple asset management in OctaneRender. You can create packages of Nodes, Nodegraphs, and Node trees, and then save them to a local drive along with thumbnails for easy identification.

To collapse the views in any of the tabs, click on the Collapse button located above the Scene Outliner tabs (Figure 2).
