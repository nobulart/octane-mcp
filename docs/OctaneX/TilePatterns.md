### Tile Patterns Parameters

Type - Determines the pattern type.

Tile Color 1 - Determines the first color used for the pattern.

Tile Color 2 - Determines the second color used for the pattern.

Line Color - Determines the color between the two tile colors.

Line Width - Determine the width of the line between the two tile colors.

UVW Transform - Positions, scales, and rotates the surface texture.

Projection - Sets how the texture projects onto the surface.
