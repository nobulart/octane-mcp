### Tripper Parameters

Primary Color - Specifies the primary color for the pattern.

Secondary Color - Specifies the secondary color for the pattern.

Tertiary Color - Specifies the tertiary color for the pattern.

Time - Provides control over animating the texture effect.

UVW Transform - Positions, scales, and rotates the surface texture.

Projection - Sets how the texture projects onto the surface.
