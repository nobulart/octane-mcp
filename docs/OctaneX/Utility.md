The Utility category in the [Textures](javascript:void(0);) node list consists of a collection of nodes that can be used for a variety of tasks such as texture and material channel operations (figure 1). This category also contains a Texture Switch node which can be used to connect two or more texture nodes to the same input pin for switching purposes during the rendering process.
