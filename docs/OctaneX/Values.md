### General Value Nodes

Bit Mask - Returns a bit value.

Bool Value - Returns a boolean value of true if enabled.

Directory Name - Specifies a directory name. 

Enum Value - Specifies an enumerator value.

File Name - Specifies a file name at a directory location.

Float Value - Specifies a float value for X, Y, Z. 

Image Resolution - Specifies an image resolution with the option to lock the aspect ratio. 

Int Value - Specifies integer values for X, Y, Z.

String Value - Spcifies a string value. 

Sun Direction - Determines the sun direction using parameters found in the Daylight Environment node.
