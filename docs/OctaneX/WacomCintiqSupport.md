OctaneRender® supports the Wacom Cintiq tablet. Installing the tablet is straightforward with the correct drivers for the tablet model and operating system. For Windows® 10 and 11 systems, disable the Use Windows Ink option from the Wacom Tablet Properties window (Figure 1).
