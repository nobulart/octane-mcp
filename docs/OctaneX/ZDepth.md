The Z Depth node generates z depth information in the form of a greyscale rendering when applied to the [Diffuse](javascript:void(0);) component of scene objects (figure 1). However, the data this texture node produces can be used in a variety of ways.

### Z Depth Parameters

Normalize Results - Remaps the data to the 0 to 1 range.

Normalization Range - The minimum and maximum values for the normalization range.
