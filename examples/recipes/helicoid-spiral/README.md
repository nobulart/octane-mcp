# Helicoid Spiral + Torus Knot (Math Grammar)

Render two parametric surfaces — a helicoid spiral and a (2,3) torus knot — with distinct blue and orange emissive materials to demonstrate combined-OBJ multi-group assignment.

## Usage

1. Import `scene.obj` with `octane_import_geometry(path="examples/recipes/helicoid-spiral/scene.obj", name="helicoid-spiral")`.
2. Create + assign materials per `usemtl` group.
3. Set camera, lighting, then `octane_save_preview`.

Camera: position [7.878, 5.712, 11.002] -> target [0.112, -0.112, 0.0], fov 50.

## Assets
